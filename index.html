My Story

Thank you for taking your time to view my resume. I want to tell you, in few words, some of my winning points into my carrier.

When I worked on Yonder (formerly Nethrom), a client (which deals with organization of business meetings) wanted to convert an Excel spreadsheet into a web application. I developed the application from scratch, I optimized the meeting times using backtracking algorithms, finally I was able to help the client to increase its profits by more than 300% in less than an year.

Other interesting project was on Smartcrew, where I worked for a telecom company. Here I developed a web application which is used today to distribute the marketing materials from the company headquarters to the marketing departments from countries where the company have a business.

Another interesting project, for the same client, was a web application where the client was able to manage routes for his agents and optimize them. On this project I worked in a team with another PHP programmer and with a few IOS and Android developers.

Another project I worked on with pleasure was one in which I used Python. The project consisted of taking information from several APIs (either SOAP or RESTful) and then comparing the data and finally obtaining a database with consistent information without duplicates (it was about more than 50 million records). Then this database became accessible via the RESTful API. This project was developed for a US client.

I would like to mention the last projects I participated in, a project for the energy industry, where I worked as a senior developer on the backend and API side (was used SOAP and REST API); another project for travel industry, where I worked as developer on the RESTful API and on the backend.

Right now I work on big project for an US company, as Senior Python Payment developer. My main role is to update and maintain the coding part for the payment and to do fraud analysis using different tools.
