My Story

Thank you for taking the time to review my resume. I would like to briefly highlight some of the key achievements in my career.

When I worked at Yonder (formerly Nethrom), a client (specializing in the organization of business meetings) wanted to convert an Excel spreadsheet into a web application. I developed the application from scratch, optimizing meeting times using backtracking algorithms. Ultimately, I helped the client increase their profits by more than 300% in less than a year.

Another interesting project was with Smartcrew, where I worked for a telecom company. Here, I developed a web application that is still used today to distribute marketing materials from the company headquarters to the marketing departments in the countries where the company operates.

For the same client, I also worked on a web application that enabled the client to manage and optimize routes for their agents. On this project, I collaborated with another PHP programmer and several iOS and Android developers.

One project I particularly enjoyed involved using Python. The project entailed gathering information from several APIs (both SOAP and RESTful), comparing the data, and creating a database with consistent, duplicate-free information (comprising over 50 million records). This database was then made accessible via a RESTful API. This project was developed for a US client.

I'd also like to mention the last few projects I participated in: a project for the energy industry, where I worked as a senior developer focusing on the backend and API side (using both SOAP and REST APIs); and another project for the travel industry, where I worked as a developer on the RESTful API and backend.

Currently, I am working on a large project for a US company as a Senior Python Payment Developer. My main responsibilities include updating and maintaining the payment coding part and conducting fraud analysis using various tools.
